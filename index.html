BabyMemo

Easily input baby log.

BabyMemoは子供の記録を簡単に録るためのアプリケーションです。


シンプルな操作で簡単に登録！
よく使う項目をウィジェットから素早く登録！


活動記録だけじゃなく、写真や日記などで子供の成長を記録します。

Download

Easily Input

わずか数タップで簡単に活動を記録することができます。記録した内容は日にちごとにタイムライン表示することにより、１日の活動を簡単に把握することができます。

It Looks Cute

シンプルなUIとかわいいアイコンで、子供の活動を記録します。

Quick Input

よく使う項目をウィジェットに表示することで、毎日入力する活動を素早く記録することができます。

Photos And Diary

写真や日記など、子供の成長を活動記録とともに残す事ができます。
